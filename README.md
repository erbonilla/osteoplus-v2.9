# Osteóplus v2.9

Responsive PWA landing-page project for Osteóplus, a Barcelona osteopathy, physiotherapy, and rehabilitation product.

## Stack

- Next.js 15 App Router
- React 19
- TypeScript strict
- Tailwind CSS v4 with semantic CSS variables
- next-intl with Spanish default and English secondary
- next-themes with class strategy
- @ducanh2912/next-pwa
- Biome, Vitest, Playwright, axe-core

## Commands

```bash
pnpm install
pnpm dev
pnpm check
pnpm typecheck
pnpm test
pnpm e2e
pnpm build
pnpm validate
```

## Source Documents

Project source documents are mirrored under `docs/project-source/`. Current visual setup follows the v4.2 token migration from `brand-kit.md` and `style-guide.md`: filled CTA Teal-700, accent Orange-700, focus ring Teal-500, and Teal-600 reserved for logo, borders, and icon emphasis.
